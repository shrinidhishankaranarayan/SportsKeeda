For Automation Scripts Plese folow belo steps.

Create a New project in eclipse and add the Sportskeeda.java file to the project.
Configure the build path using Junit and POI jar files.
Change the path of the Web driver in System.setproperty according to the local machine
Change the path of the excel sheet according to the path saved.
Change the Last column From TRUE to FALSE and vice versa to validate various scenarios.
Note: At any given time only one row can have TRUE at the nth column of the particular row.
